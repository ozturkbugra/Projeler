Sıkça Sorulan Sorular

Merak ettiğiniz veya edebileceğiniz soruları cevaplayarak kafanızdaki soru işaretlerini ortadan kaldırdık.

Öğrencilerin proje derslerinden geçebilmeleri için temel CRUD işlemlerini (Ekleme,Okuma,Güncelleme,Silme) yapabilen projeler yapabiliyoruz. Masaüstü için C# Windows Form, Web için ASP.NET, PHP dilleri ile projeler hazırlıyoruz.

Hafta içi 18'den sonra, Cumartesi günü 13'den sonra, Pazar günü ise tüm gün benle iletişime geçebilirsiniz.

Bu konunun ucu açık çünkü Web Tabanlı bir site istiyorsanız sunucu kalitesine alanına göre değişir. Öğrenciler için bütçelerini zorlamayacak bir fiyata yapıyorum.
